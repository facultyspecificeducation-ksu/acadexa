# Acadexa

Intelligent Academic Advising Desktop Application.

Monorepo containing:
- apps/desktop  - Electron shell (main + preload)
- apps/web      - React renderer (Vite + MUI, RTL Arabic support)
- apps/api      - FastAPI backend (Expert System, AI layer, data processing)
- packages/*    - Shared types, config and UI kit consumed across apps

See docs/ for architecture notes and ADRs.
